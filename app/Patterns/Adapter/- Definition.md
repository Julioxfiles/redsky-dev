
# Adapter Pattern

## Definition

The Adapter Pattern is a structural design pattern in object-oriented programming that acts as a bridge between two incompatible interfaces.

## Where it is used:

This is especially useful when integrating legacy code, third-party libraries, or external APIs that do not follow the structure of the application.
